About calco

calco is a high-performance, cross-platform math library written in C and exposed as a Python extension.

It provides fast, accurate, and comprehensive implementations of advanced mathematical operations beyond what's available in Python's math or numpy modules.

🚀 Features

⚡

Fast C-level execution

Leverage the power of C for unparalleled performance in your mathematical computations.

🧮

60+ Math Operations

A comprehensive suite covering arithmetic, trigonometry, logarithms, exponentials, and special functions.

🧩

Cross-Platform Portable

Designed to work seamlessly across Windows, Linux, and macOS environments.

📦

Direct Python Import

Packaged as .pyd / .so for easy integration into your Python projects.

📦 Installation

Install calco from PyPI:

pip install calco

Or install directly from a wheel file (e.g., calco-1.0.0-cp311-win_amd64.whl):

pip install calco-1.0.0-cp311-win_amd64.whl

🧪 Example Usage

import calco

# Basic math
print(calco.add(3.0, 4.5))            # 7.5
print(calco.divide(10.0, 3.0))        # 3.333...

# Trigonometry
print(calco.sine(1.57))               # ≈ sin(90°)
print(calco.arctangent2(2.0, 3.0))    # atan2(y, x)

# Logarithmic
print(calco.log_base10(1000))         # 3.0

# Special
print(calco.gamma_function(0.5))      # ≈ sqrt(pi)
print(calco.fused_multiply_add(2, 3, 4))  # (2*3)+4 = 10
