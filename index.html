About calco

calco is a high-performance, cross-platform math library written in C and exposed as a Python extension.

It provides fast, accurate, and comprehensive implementations of advanced mathematical operations beyond what's available in Python's MATH modules. calco is designed to be the fastest library for complex or moderately complex individual mathematical operations in Python, offering speeds over 30% faster than the built-in math module. Additionally, it supports ready-to-use functions not available in math.

Important Notes for Installation

Important Note about Version: Please be aware that version 1.0.0 of the `calco` library is the recommended stable release. Other versions might be unstable.

Renaming the .pyd file: After downloading and installing the Wheel (.whl) file, you might need to rename the .pyd (or .so on Linux/macOS) file located inside your Python's site-packages folder. The file should be named calco.pyd (or calco.so) to allow direct importing using import calco.

If you do not rename the file, you will not be able to import the library using the conventional import calco. In this case, you will need to import it using the full name of the `pyd` file, by replacing hyphens (`-`) with underscores (`_`) and removing `.pyd` from the end. For example: 

# Example if you don't rename the .pyd file
# (Replace 'calco_1_0_0_cp311_win_amd64' with the actual full name of your .pyd file)
import calco_1_0_0_cp311_win_amd64
print(calco.add(1, 2))

How to find your site-packages folder: You can find the path to your site-packages folder using the following commands in your command prompt/terminal: 

# For Windows
python -c "import site; print(site.getsitepackages())"

# For Linux/macOS
python3 -c "import site; print(site.getsitepackages())"

Importing after Installation: After ensuring that the calco.pyd (or calco.so) file is correctly named and located in your site-packages folder, you can simply import and use the library: 

import calco

# You can now use all calco functions, e.g.
print(calco.add(5.0, 2.5))
üß™ Example Usage
import calco

# Basic math
print(calco.add(3.0, 4.5))            # 7.5
print(calco.divide(10.0, 3.0))        # 3.333...

# Trigonometry
print(calco.sine(1.57))               # ‚âà sin(90¬∞)
print(calco.arctangent2(2.0, 3.0))    # atan2(y, x)

# Logarithmic
print(calco.log_base10(1000))         # 3.0

# Special
print(calco.gamma_function(0.5))      # ‚âà sqrt(pi)
print(calco.fused_multiply_add(2, 3, 4))  # (2*3)+4 = 10
